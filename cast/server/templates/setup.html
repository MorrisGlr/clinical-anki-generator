OpenAI API Key Setup

CAST uses OpenAI to generate the enriched explanations on your flashcards. You need an API key to get started.

What is an API key?

It is a password-like code that lets CAST talk to OpenAI on your behalf. You create it once in your OpenAI account, then paste it here. CAST saves it to a local .env file on your computer — it is never sent anywhere else.

How to get your API key

  1. Open platform.openai.com in your browser and sign in (or create a free account).
  2. Click your profile icon in the top-right corner, then click API keys.
  3. Click Create new secret key. Give it a name like CAST.
  4. Copy the key — it starts with sk-. This is the only time it will be shown.
  5. Paste it into the field below and click Save.
Note on cost: API calls are billed by OpenAI based on usage. A typical batch of 40 cards costs roughly $0.40–$4.00. You can set a monthly spending limit at platform.openai.com → Settings → Billing → Usage limits.
